# StyleBot

Telegram-бот на Python для подбора повседневных образов, рекомендаций под событие и списков вещей в поездку. Рекомендации строятся на **правилах и ключевых словах** (без внешних AI API). Данные пользователей хранятся в **JSON-файле** `data/users.json`.

## Требования

- Python **3.10+** (рекомендуется актуальная стабильная версия)
- Токен бота от [@BotFather](https://t.me/BotFather)

## Установка

```bash
cd stylebot
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка

1. Скопируйте пример переменных окружения:

   ```bash
   cp .env.example .env
   ```

2. Откройте `.env` и вставьте токен:

   ```env
   TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН
   ```

## Запуск

```bash
source .venv/bin/activate
python bot.py
```

При первом сохранении данных будет создан каталог `data/` и файл `data/users.json`.

## Команды

| Команда | Описание |
|--------|----------|
| `/start` | Приветствие и возможности бота |
| `/help` | Список команд |
| `/look` | Подсказка, как запросить образ |
| `/trip` | Подсказка по чемодану |
| `/profile` | Показать сохранённые предпочтения и гардероб |
| `/setstyle` | Выбрать стиль кнопками |
| `/setcolors` | Формат сообщения для цветов |
| `/wardrobe` | Как добавить вещи в гардероб |
| `/reset` | Очистить сохранённые данные |
| `/examples` | Примеры фраз |

Свободный текст обрабатывается эвристиками: по ключевым словам выбирается сценарий «образ», «поездка», «сохранить предпочтения» или «гардероб».

## Структура проекта

```
stylebot/
  bot.py              # код бота
  requirements.txt    # зависимости
  .env.example        # пример конфигурации
  README.md           # документация
  data/               # создаётся при работе
    users.json        # профили пользователей, гардероб, история
```

## Замечания для лабораторной

- Токен **не** хранится в коде — только в переменной окружения `TELEGRAM_BOT_TOKEN` (через `.env` и `python-dotenv`).
- Ошибки логируются в консоль; пользователю отправляются короткие дружелюбные сообщения.
- Логику можно расширять: добавлять словари ключевых слов и шаблоны ответов в `bot.py`.
