# 🧠 Описание интеграции

В рамках лабораторной работы был улучшен Telegram-бот StyleBot за счет интеграции с внешним источником данных — API погоды Open-Meteo.

Бот был доработан таким образом, чтобы получать актуальные данные о погоде по названию города и использовать их для формирования рекомендаций по одежде.

---

# 🌐 Источник данных

## Выбранный источник:
Open-Meteo API

- Geocoding API — для получения координат города
- Weather API — для получения текущей температуры

## Почему выбран:

- бесплатный API без необходимости регистрации
- простая структура запросов
- поддержка поиска городов по названию
- подходит для учебного проекта

# ⚙️ Реализация

## Как работает интеграция

Пользователь вводит запрос:

- "погода в Москве"  
- "что надеть в Париже"  

Бот:

- извлекает название города  
- нормализует его (убирает падежи и лишние слова)  
- получает координаты через Geocoding API  
- запрашивает погоду через Weather API  
- формирует ответ

def normalize_city_name(city: str) -> str:
    city = city.strip().lower()

    for word in ["завтра", "сегодня", "через неделю"]:
        city = city.replace(word, "")

    city = " ".join(city.split())

    if city.endswith("е"):
        city = city[:-1]
    elif city.endswith("и"):
        city = city[:-1] + "ь"
    elif city.endswith("у"):
        city = city[:-1] + "а"

    return city
