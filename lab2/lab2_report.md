# 🧠 Описание интеграции

В рамках лабораторной работы был улучшен Telegram-бот StyleBot за счет интеграции с внешним источником данных — API погоды Open-Meteo.

Бот был доработан таким образом, чтобы получать актуальные данные о погоде по названию города и использовать их для формирования рекомендаций по одежде.

---

# 🌐 Источник данных

## Выбранный источник:
Open-Meteo API

- Geocoding API — для получения координат города
- Weather API — для получения текущей температуры

## Почему выбран:

- бесплатный API без необходимости регистрации
- простая структура запросов
- поддержка поиска городов по названию
- подходит для учебного проекта
