# 🧠 Описание задачи

В рамках лабораторной работы был разработан Telegram-бот **StyleBot**, который помогает пользователю принимать решения в области одежды.

Бот выполняет следующие функции:
- подбор повседневного образа
- рекомендации одежды под погоду и событие
- помощь в сборе чемодана в поездку
- сохранение предпочтений пользователя

Бот взаимодействует с пользователем через текстовые сообщения и анализирует запросы, чтобы предложить структурированные рекомендации.

---

# 🎯 Какую проблему решает бот

Многие пользователи сталкиваются с проблемой выбора одежды:
- не знают, что надеть в конкретной ситуации
- не учитывают погодные условия
- испытывают сложности при сборе вещей в поездку
- тратят время на принятие решений

Бот решает эти проблемы, предлагая:
- готовые образы
- структурированные списки вещей
- рекомендации на основе контекста

## Финальный промпт

Создай Telegram-бота на Python с использованием библиотеки python-telegram-bot.

Бот называется StyleBot. Он помогает пользователю принимать решения в области одежды: подбирать образ на день, рекомендовать одежду под погоду и событие, а также собирать чемодан в поездку.

Функционал бота:
- Подбор образа по текстовому запросу пользователя
- Учет погоды, события и стиля
- Помощь в сборе чемодана по длительности и типу поездки
- Сохранение предпочтений пользователя
- Работа со свободным текстом
- Показ сохраненного профиля пользователя

Команды:
- /start
- /help
- /look
- /trip
- /profile
- /setstyle
- /setcolors
- /wardrobe
- /reset
- /examples

Как бот должен отвечать:
- простым и понятным языком
- дружелюбно
- структурированно
- с разделением ответа по категориям
- с коротким объяснением рекомендаций

Какие данные хранить:
- user_id
- username
- preferred_style
- favorite_colors
- disliked_colors
- gender
- notes
- список вещей пользователя
- по желанию историю запросов

Требования:
- бот должен быть простым и понятным
- код должен быть хорошо прокомментирован
- использовать файл для хранения данных (JSON или SQLite)
- добавить обработку ошибок
- использовать переменные окружения для токена
- не использовать сложные внешние AI API
- реализовать логику через правила, ключевые слова и шаблоны

Создай:
1. Файл bot.py с кодом бота
2. Файл requirements.txt с зависимостями
3. Файл README.md с инструкцией по запуску
4. Файл .env.example для примера конфигурации

# ⚠️ Трудности и решения

## Проблема 1:
Ошибка запуска бота:
RuntimeError: There is no current event loop

### Решение:
Проблема была связана с версией Python (3.14).  
Решение — использование стабильной версии Python (3.12).

---

## Проблема 2:
Ошибка с pip (command not found)

### Решение:
Использование команды pip3 вместо pip.

---

## Проблема 3:
Бот не запускался из-за отсутствия токена

### Решение:
Создан файл `.env` и добавлена переменная TELEGRAM_BOT_TOKEN.
